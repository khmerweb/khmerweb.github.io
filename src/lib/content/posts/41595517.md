<p>From Microsoft team</p><p>&nbsp;</p><p><strong>Summary</strong>: in this tutorial, you will learn about the TypeScript if...else statement.</p><p>&nbsp;</p><p><span style="font-size:21px;"><strong>TypeScript if statement</strong></span></p><p>&nbsp;</p><p>An if statement executes a statement based on a condition. If the condition is truthy, the if statement will execute the statements inside its body:</p><pre><code class="typescript">if(condition) {
   // if-statement
}</code></pre><p>&nbsp;</p><p>For example, the following statement illustrates how to use the if statement to increase the counter variable if its value is less than the value of the max constant:</p><pre><code class="typescript">const max = 100;
let counter = 0;

if (counter &lt; max) {
    counter++;
}

console.log(counter); // 1</code></pre><p>&nbsp;</p><p>Output:</p><pre><code>1</code></pre><p>&nbsp;</p><p>In this example, because the counter variable starts at zero, it is less than the max constant. The expression counter &lt; max evaluates to true therefore the if statement executes the statement counter++.</p><p>&nbsp;</p><p>Let’s initialize the counter variable to 100:</p><pre><code class="typescript">const max = 100;
let counter = 100;

if (counter &lt; max) {
    counter++;
}

console.log(counter); // 100</code></pre><p>&nbsp;</p><p>Output:</p><pre><code>100</code></pre><p>&nbsp;</p><p>In this example, the expression counter &lt; max evaluates to false. The if statement doesn’t execute the statement counter++. Therefore, the output is 100.</p><p>&nbsp;</p><p><span style="font-size:21px;"><strong>TypeScript if…else statement</strong></span></p><p>&nbsp;</p><p>If you want to execute other statements when the condition in the if statement evaluates to false, you can use the if...else statement:</p><pre><code class="typescript">if(condition) {
   // if-statements
} else {
  // else statements;
}</code></pre><p>&nbsp;</p><p>The following illustrates an example of using the if..else statement:</p><pre><code class="typescript">const max = 100;
let counter = 100;

if (counter &lt; max) {
    counter++;
} else {
    counter = 1;
}

console.log(counter);</code></pre><p>&nbsp;</p><p>Output:</p><pre><code>1</code></pre><p>&nbsp;</p><p>In this example, the expression counter &lt; max evaluates to false therefore the statement in the else branch executes that resets the counter variable to 1.</p><p>&nbsp;</p><p><span style="font-size:21px;"><strong>Ternary operator ?:</strong></span></p><p>&nbsp;</p><p>In practice, if you have a simple condition, you can use the ternary operator ?: rather than the if...else statement to make code shorter like this:</p><pre><code class="typescript">const max = 100;
let counter = 100;

counter &lt; max ? counter++ : counter = 1;

console.log(counter);</code></pre><p>&nbsp;</p><p><span style="font-size:21px;"><strong>TypeScript if…else if…else statement</strong></span></p><p>&nbsp;</p><p>When you want to execute code based on multiple conditions, you can use the if...else if...else statement.</p><p>&nbsp;</p><p>The if…else if…else statement can have one or more else if branches but only one else branch.</p><p>&nbsp;</p><p>For example:</p><pre><code class="typescript">let discount: number;
let itemCount = 11;

if (itemCount &gt; 0 &amp;&amp; itemCount &lt;= 5) {
    discount = 5;  // 5% discount
} else if (itemCount &gt; 5 &amp;&amp; itemCount &lt;= 10) {
    discount = 10; // 10% discount 
} else {
    discount = 15; // 15%
}

console.log(`You got ${discount}% discount. `)</code></pre><p>&nbsp;</p><p>Output:</p><pre><code>You got 15% discount. </code></pre><p>&nbsp;</p><p>This example used the if...else if...else statement to determine the discount based on the number of items.</p><p>&nbsp;</p><p>If the number of items is less than or equal to 5, the discount is 5%. The statement in the if branch executes.</p><p>&nbsp;</p><p>If the number of items is less than or equal to 10, the discount is 10%. The statement in the else if branch executes.</p><p>&nbsp;</p><p>When the number of items is greater than 10, the discount is 15%. The statement in the else branch executes.</p><p>&nbsp;</p><p>In this example, the assumption is that the number of items is always greater than zero. However, if the number of items is less than zero or greater than 10, the discount is 15%.</p><p>&nbsp;</p><p>To make the code more robust, you can use another else if instead of the else branch like this:</p><pre><code class="typescript">let discount: number;
let itemCount = 11;

if (itemCount &gt; 0 &amp;&amp; itemCount &lt;= 5) {
  discount = 5; // 5% discount
} else if (itemCount &gt; 5 &amp;&amp; itemCount &lt;= 10) {
  discount = 10; // 10% discount
} else if (itemCount &gt; 10) {
  discount = 15; // 15%
} else {
  throw new Error('The number of items cannot be negative!');
}

console.log(`You got ${discount}% discount. `);
</code></pre><p>&nbsp;</p><p>Output:</p><pre><code>You got 15% discount. </code></pre><p>&nbsp;</p><p>In this example, when the number of items is greater than 10, the discount is 15%. The statement in the second else if branch executes.</p><p>&nbsp;</p><p>If the number of items is less than zero, the statement in the else branch executes.</p><p>&nbsp;</p><p>Summary</p><p>&nbsp;</p><ul><li>Use the if statement to execute code based on a condition.</li><li>Use the else branch if you want to execute code when the condition is false. It’s good practice to use the ternary operator ?: instead of a simple if…else statement.</li><li>Use if else if...else statement to execute code based on multiple conditions.</li></ul>
